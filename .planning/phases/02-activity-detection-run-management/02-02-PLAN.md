    **State enum:**
    ```swift
    enum ClassifierState: Sendable { case idle, chairlift, skiing }
    ```

    **ActivityClassifier actor:**
    - Private stored state: `state: ClassifierState = .idle`
    - `skiingOnsetSeconds: Double = 3.0`, `runEndSeconds: Double = 2.0`
    - `pendingSkiingOnsetAt: Date? = nil` — nil means not accumulating
    - `pendingRunEndAt: Date? = nil`
    - `pendingFrames: [FilteredFrame] = []` — held during onset window
    - `currentRunID: UUID? = nil`
    - `varianceWindow: [Double] = []`, `varianceWindowSize: Int = 50`
    - `latestGPS: GPSReading? = nil`
    - `latestActivity: CMMotionActivity? = nil`
    - `private var consumptionTasks: [Task<Void, Never>] = []`

    **startDay(frameStream:gpsStream:activityStream:persistenceService:):**
    Sets state to `.chairlift` (default safe state on arm). Launches three Tasks:
    - Task 1: `for await gps in gpsStream { latestGPS = gps }`
    - Task 2: `for await act in activityStream { latestActivity = act }`
    - Task 3: `for await frame in frameStream { await processFrame(frame) }`
    Store all three in consumptionTasks.

    **endDay():**
    Cancel consumptionTasks. If state == .skiing, finalize open RunRecord:
    `try? await persistenceService?.finalizeRunRecord(runID: currentRunID!, endTimestamp: Date())`
    Reset all state to idle.

    **processFrame(_ frame: FilteredFrame):**
    1. Update variance window: compute g = sqrt(x²+y²+z²) from userAccelX/Y/Z, append, cap at varianceWindowSize (removeFirst if overflow).
    2. Let now = Date()
    3. Switch on state:
       - `.chairlift`: call `evaluateSkiingOnset(frame: frame, now: now)`
       - `.skiing`: call `evaluateRunEnd(now: now)`
       - `.idle`: do nothing

    **evaluateSkiingOnset(frame:now:):**
    If skiingSignalActive():
      - If pendingSkiingOnsetAt == nil: set pendingSkiingOnsetAt = now
      - Elapsed = now.timeIntervalSince(pendingSkiingOnsetAt!)
      - hysteresisProgress = min(elapsed / skiingOnsetSeconds, 1.0)
      - If elapsed >= skiingOnsetSeconds: confirmSkiingTransition()
      - Else: append frame to pendingFrames
    Else:
      - Reset pendingSkiingOnsetAt = nil, pendingFrames = []

    **confirmSkiingTransition():**
    - Let runID = UUID(); currentRunID = runID
    - Let startTimestamp = pendingFrames.first?.timestamp.toDate() ?? Date()
      (FilteredFrame.timestamp is TimeInterval — convert: Date(timeIntervalSince1970: frame.timestamp) or use Date() if pendingFrames is empty)
    - Call `try? await persistenceService?.createRunRecord(runID: runID, startTimestamp: startTimestamp)`
    - state = .skiing; pendingSkiingOnsetAt = nil; pendingFrames = []

    **evaluateRunEnd(now:):**
    If chairliftSignalActive(at: now):
      - If pendingRunEndAt == nil: set pendingRunEndAt = now
      - If now.timeIntervalSince(pendingRunEndAt!) >= runEndSeconds: confirmChairliftTransition()
    Else:
      - Reset pendingRunEndAt = nil

    **confirmChairliftTransition():**
    - If let runID = currentRunID: `try? await persistenceService?.finalizeRunRecord(runID: runID, endTimestamp: Date())`
    - currentRunID = nil; state = .chairlift; pendingRunEndAt = nil

    **chairliftSignalActive(at now: Date) -> Bool:**
    - isAutomotive: latestActivity?.automotive == true && latestActivity?.confidence != .low
    - GPS blackout = latestGPS == nil || (latestGPS?.horizontalAccuracy ?? -1) < 0
    - If GPS blackout and state == .chairlift: speedInLiftRange = true (sustain on blackout)
    - Else if GPS valid: speedInLiftRange = gpsSpeed >= 0.5 && gpsSpeed <= 7.0
    - Else: speedInLiftRange = false
    - lowVariance: gForceVariance < 0.01
    - Return isAutomotive && speedInLiftRange && lowVariance

    **skiingSignalActive() -> Bool:**
    - GPS valid: gpsSpeed >= 3.0 (else if GPS unavailable: speedOK = true)
    - highVariance: gForceVariance > 0.005
    - notAutomotive: !(latestActivity?.automotive ?? false) || latestActivity?.confidence == .low
    - Return speedOK && highVariance && notAutomotive

    **gForceVariance: Double** (computed property):
    guard varianceWindow.count > 1; mean = sum/count; return sumSq / (count - 1)
